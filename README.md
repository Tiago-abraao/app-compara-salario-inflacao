## Aplicação que faz a comparação entre salário e inflação.

Desenvolvi uma aplicação de terminal utilizando JavaScript (Node.js) e a biblioteca readline-sync para consulta e análise de dados do salário mínimo e da inflação no Brasil entre os anos de 2010 e 2020.

Funcionalidades implementadas:

✅ Listagem dos salários mínimos de 2010 a 2020
✅ Exibição do índice de inflação (IPCA) de cada ano
✅ Comparação entre o percentual de aumento do salário mínimo e a inflação do período

Conceitos praticados:

-Estruturas de dados com arrays de objetos
-Laços de repetição (for)
-Condicionais (if, else if, else)
-Manipulação e formatação de strings
-Formatação de valores monetários e percentuais
-Cálculos de variação percentual
-Entrada de dados pelo usuário via terminal
-Organização de lógica de programação em Node.js

O projeto permite visualizar não apenas a evolução do salário mínimo ao longo dos anos, mas também analisar se os reajustes acompanharam os índices de inflação, fornecendo uma comparação simples e objetiva dos dados.
Ao iniciar a aplicação é mostrado ao usuário um menu com 3 opções:
A 1 lista os salários mínimos de 2010 à 2020.
A 2 lista o índice IPCA de 2010 à 2020.
A 3 faz a Comparação entre o percentual de aumento salarial e o IPCA.
Se o usuário tentar escolher uma opção fora das apresentadas, uma mensagem de opção inválida é exibida.

An application that compares salary growth and inflation.
I developed a terminal-based application using JavaScript (Node.js) and the readline-sync library to query and analyze Brazil’s minimum wage and inflation data between 2010 and 2020.
Implemented Features:
✅ Display of minimum wages from 2010 to 2020
✅ Display of the annual inflation rate (IPCA)
✅ Comparison between the minimum wage increase percentage and the inflation rate for the same period
Concepts Practiced:
Data structures using arrays of objects
Loops (for)
Conditional statements (if, else if, else)
String manipulation and formatting
Currency and percentage formatting
Percentage change calculations
User input through the terminal
Organization of programming logic in Node.js
This project allows users not only to visualize the evolution of the minimum wage over the years but also to analyze whether wage adjustments kept pace with inflation, providing a simple and objective comparison of the data.
When the application starts, the user is presented with a menu containing three options:
1️⃣ Display minimum wages from 2010 to 2020.
2️⃣ Display IPCA inflation rates from 2010 to 2020.
3️⃣ Compare salary increase percentages with IPCA inflation rates.
If the user selects an option outside the available choices, an "Invalid Option" message is displayed.
